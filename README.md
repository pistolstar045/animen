アプリ名：animen

## サイト概要

### サイトテーマ
・「アニメ」に特化したコミュニティサイト
・自分の好きなアニメを他の人と共有できる場

### テーマを選んだ理由
・作者自身もアニメが好きで、アニメを通した交流の場を作りたかったから
・コミュニティサイト自体の作成をしてみたかったため

### ターゲットユーザ
・アニメが好きな人
・アニメを通して交流する場を求めている人
・アニメを好きな者同士でグループを作り、イベントやグループの
　活動を楽しみたい人

### 主な利用シーン
・自分の好きなアニメを誰かと共有したいとき
・アニメという共通の趣味を持つ人と交流したいとき
・アニメ好きな人とグループを作りたいとき
